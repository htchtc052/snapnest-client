# Snapnest Client

Клиент Nuxt 4 для Snapnest: приватная фотобиблиотека с альбомами и публичными ссылками.

## Стек
- Nuxt 4 + Vue 3
- `@nuxt/ui`
- Pinia
- `nuxt-auth-sanctum`

## Окружение
- `NUXT_PUBLIC_API_BASE` — базовый URL API (по умолчанию `http://localhost:8000`)
- `NUXT_PUBLIC_SITE_URL` — базовый URL фронта для share-ссылок (по умолчанию `http://localhost:3000`)

## Роуты
- `/`
  - Лендинг для гостей.
- `/login`
  - Форма входа.
- `/register`
  - Форма регистрации.
- `/account`
  - Основной таймлайн фото.
  - Infinite scroll с серверной пагинацией (для сценария большого архива).
  - Мультивыбор: добавить в альбом, поделиться, переименовать, переместить в корзину.
- `/account/albums`
  - Список альбомов с созданием/редактированием/удалением.
- `/account/albums/:id`
  - Просмотр альбома и удаление выбранных фото из альбома.
- `/account/trash`
  - Корзина: восстановление или перманентное удаление выбранных фото.
- `/albums/:token`
  - Публичная страница альбома по токену (read-only).

## Ограничение текущего MVP
- Сейчас публичный шаринг реализован только для альбомов.
