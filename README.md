# Snapnest Client

Клиент Nuxt 4 для Snapnest: приватная фотобиблиотека с публичными шарами.

## Стек
- Nuxt 4 + Vue 3
- `@nuxt/ui`
- Pinia
- `nuxt-auth-sanctum`

## Окружение
- `NUXT_PUBLIC_API_BASE` — базовый URL API (по умолчанию `http://localhost:8000`)
- `NUXT_PUBLIC_SITE_URL` — базовый URL фронта для share-ссылок (по умолчанию `http://localhost:3000`)

## Роуты
- `/`
  - Лендинг для гостей.
- `/login`
  - Форма входа.
- `/register`
  - Форма регистрации.
- `/account`
  - Основной таймлайн фото.
  - Пагинация, viewer, мультивыбор, создание альбома, переименование фото, перемещение в корзину.
- `/account/albums`
  - Список своих альбомов.
- `/account/trash`
  - Корзина: восстановление или перманентное удаление выбранных фото.
- `/albums/:token`
  - Публичный альбом по токену, viewer и скачивание оригиналов.

## Ограничение текущего MVP
- Сейчас основной collaborative flow строится вокруг альбомов, а не вокруг ручного управления альбомами.
